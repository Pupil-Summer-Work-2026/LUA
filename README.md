# LUA: dev branch
## This branch is used for the main development of LUA website. All pull requests that consist of code changes must target this branch.

## Build & Run
To build this project, you will need:
- Node.js >=24.18.0

1. Install the Node.js and restart your IDE
2. In your IDEs' terminal run <npm.cmd install> from projects' root folder
3. After modules finish installing run <npm.cmd run dev>. This will open the webpage in your default browser.

## UPD 13.07.26
As of now, this project uses Vite build tool, which allows changes to be hot-loaded. That means that you do not need to restart the server to apply changes to the code. To hot-load the changes:
1. Save them to the file (CTRL+S)
2. Reload the page in your browser (it is possible that the browser will do that automatically)
